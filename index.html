About WrzasqPl CForm

WrzasqPl-CForm is set of AWS CloudFormation extensions.

Project Modules

This project has declared the following modules:

Name Description
Commons library CForm generic helpers.
WrzasqPl::AppSync::GraphQlApi data provider AWS CloudFormation data provider for AppSync APIs.
WrzasqPl::Cognito::Domain data provider AWS CloudFormation data provider for Cognito domain.
CloudFormation macro processor. AWS CloudFormation template processing macro.
WrzasqPl::AWS::Account resource AWS CloudFormation resource handler for account entity.
WrzasqPl::AWS::Organization resource AWS CloudFormation resource handler for organizations.
WrzasqPl::AWS::OrganizationalUnit resource AWS CloudFormation resource handler for organizational units.
WrzasqPl::AWS::PasswordPolicy resource AWS CloudFormation resource handler for account password policy.
